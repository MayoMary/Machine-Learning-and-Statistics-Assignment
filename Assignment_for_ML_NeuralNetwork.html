Data Scaling¶

https://medium.com/greyatom/why-how-and-when-to-scale-your-features-4b30ab09db5e

Dataset often contain features highly varying in magnitudes, units and range. This is a propblem because most of the machine learning algorithms use Eucledian distance between two data points in their computations. If left alone, these algorithms only take in the magnitude of features neglecting the units. The results would vary greatly between different units, 5kg and 5000gms. The features with high magnitudes will weigh in a lot more in the distance calculations than features with low magnitudes. This is the case in Boston dataset as well as seen from the cells above. Therefore, we do data scaling first before fitting the neural network model.
